## English

**Merlin's Vetum Librem** (*Vetus Librem · Liber I*) is a fractal cosmological codex — a personal, cross-cultural exploration of recurring questions about origin, consciousness, and form.

It weaves together science, mythology, psychology, and poetry into a single structural map: the **Nullpunkt** (origin), **Dualität** (light and shadow), **Grenze** (boundary), **Zeit** (time), and the **Fraktal** (self-similar pattern at every scale). The same mechanics appear under different names across traditions — from Eden and Yin/Yang to Sköll & Hati and quantum observation — and all paths are meant to converge back to the same ground.

The repository holds the source texts, a static HTML reader, and an archive of the original scattered drafts.

### Contents

| Section | Description |
|---------|-------------|
| **Arch & Tectonics** | Structural map of the codex and its path library |
| **Prolog — Das Licht** | Universal entry: Nullpunkt, duality, time, consciousness, fractal |
| **Liber I — De Fundamentis** | Foundations: consciousness, word, language, collapse |
| **Kapitel I–V** | Core chapters: *Die Grenze*, *Die Dualität*, *Das Werkzeug*, *Die Zeit*, *Das Fraktal* |
| **Caput X–XXII** | Latin chapters deepening specific themes (radii, aether, fire, Norse myth) |
| **Intermezzo** | *Der Apfel, der keiner war* — bridge toward Loki |
| **Extras** | Runa mantra and draft sketches (Depidgeons) |

### Repository layout

```
content/          Canonical Markdown sources (read in order)
site/             Static HTML site (generated)
tools/build.py    Builds site/ from content/
archive/          Original files before reorganization
```

### Reading & building

**Recommended reading order:** Prolog → Fundamentos → Núcleo (Kapitel I–V) → Capítulos → Intermezzo

Open [`site/index.html`](site/index.html) in a browser, or regenerate the site:

```powershell
python tools/build.py
```

Requires Python 3. No external dependencies.
